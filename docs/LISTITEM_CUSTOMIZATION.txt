================================================================================
                LISTITEM - ÖZELLEŞTİRME REHBERİ
================================================================================

🎨 ListItem artık tamamen özelleştirilebilir! Her alan değiştirilebilir.

================================================================================
1. YENİ ÖZELLEŞTİRİLEBİLİR PARAMETRELER
================================================================================

LEADING ITEM (Sol):
├── leadingSize: number (varsayılan: 40)
├── leadingBorderRadius: number (varsayılan: 8)
├── leadingIconSize: number (varsayılan: 24)
└── leadingBackgroundColor: string (varsayılan: '#E8F5E9')

TEXT CONTENT:
├── titleStyle: object (özel title stili)
└── supportingStyle: object (özel supporting text stili)

TRAILING ITEM (Sağ):
├── trailingIconSize: number (varsayılan: 24)
└── trailingIconColor: string (otomatik: dark mode'a göre)

CONTAINER:
├── containerStyle: object (özel container stili)
└── backgroundColor: string (özel arkaplan rengi)

================================================================================
2. LEADING ITEM ÖZELLEŞTİRME
================================================================================

// Daha büyük leading item
<ListItem 
  leadingType="icon"
  leadingIcon={require('./icon.png')}
  leadingSize={56}              // 56x56px
  leadingBorderRadius={16}      // Daha yuvarlak
  leadingIconSize={32}          // Daha büyük ikon
  leadingBackgroundColor="#FFE8E8"
  titleText="Büyük İkon"
/>

// Kare leading item
<ListItem 
  leadingType="image"
  leadingImage={require('./profile.png')}
  leadingSize={48}
  leadingBorderRadius={0}       // Kare (köşesiz)
  titleText="Kare Profil"
/>

// Tam yuvarlak (circle)
<ListItem 
  leadingType="icon"
  leadingIcon={require('./heart.png')}
  leadingSize={40}
  leadingBorderRadius={20}      // Yarısı = tam yuvarlak
  leadingBackgroundColor="#E8F5FF"
  titleText="Yuvarlak İkon"
/>

// Gradient background (ViewGradient component ile)
<ListItem 
  leadingType="icon"
  leadingIcon={require('./star.png')}
  leadingSize={48}
  leadingBorderRadius={12}
  leadingBackgroundColor="linear-gradient(135deg, #667eea 0%, #764ba2 100%)"
  titleText="Gradient Arka Plan"
/>

================================================================================
8. KULLANIM İPUÇLARI
================================================================================

✅ DO (Yapın):
1. leadingSize ve leadingBorderRadius'u birlikte kullanın
2. Yuvarlak için: borderRadius = size / 2
3. titleStyle ve supportingStyle ile tutarlı font ailesi kullanın
4. trailingIconColor ile dark mode uyumlu renkler seçin
5. containerStyle ile gölge ve border'ı birlikte kullanabilirsiniz

❌ DON'T (Yapmayın):
1. Çok büyük leadingSize kullanmayın (max 64px önerilir)
2. Çok fazla farklı renk kullanmayın (max 2-3 renk)
3. titleStyle fontSize'ı çok büyük yapmayın (max 24px)
4. Container'a çok fazla özel stil eklemeyin (karmaşık olur)
