# CARDPOST COMPONENT KULLANIM KILAVUZU

## GENEL BAKIŞ
CardPost component'i sosyal medya benzeri gönderi kartları oluşturmak için kullanılır. 
Farklı içerik türlerini destekler ve kullanıcı etkileşimleri için callback fonksiyonları sağlar.

## IMPORT
```javascript
import CardPost from '../../components/CardPost';
```

## TEMEL KULLANIM

### Basit Text Gönderisi
```javascript
<CardPost
  name="Sarah"
  time="2m ago"
  text="One week done 🎉! Food already tastes better and I'm breathing easier."
  type="Text"
  likes="1.4K"
  comments="128"
  onLike={() => console.log('Liked')}
  onComment={() => console.log('Commented')}
  onSave={() => console.log('Saved')}
  onMore={() => console.log('More options')}
/>
```

## PROPS DETAYLARI

### Zorunlu Props
Yok - Tüm props'lar opsiyoneldir ve varsayılan değerleri vardır.

### Opsiyonel Props

#### Temel Bilgiler
- `name` (string): Kullanıcı adı (varsayılan: "Sarah")
- `time` (string): Gönderi zamanı (varsayılan: "2m ago")
- `avatar` (image source): Kullanıcı avatar'ı (varsayılan: avatar2.png)
- `text` (string): Gönderi metni
- `type` (string): Gönderi türü (varsayılan: "Text")

#### İstatistikler
- `likes` (string): Beğeni sayısı (varsayılan: "1.4K")
- `comments` (string): Yorum sayısı (varsayılan: "128")

#### Tema
- `darkMode` (boolean): Karanlık tema (varsayılan: false)

#### Callback Fonksiyonları
- `onLike` (function): Beğeni butonuna basıldığında
- `onComment` (function): Yorum butonuna basıldığında
- `onSave` (function): Kaydet butonuna basıldığında
- `onMore` (function): Daha fazla seçenek butonuna basıldığında

## CALLBACK FONKSİYONLARI

### Beğeni Fonksiyonu
```javascript
const handleLike = (postId) => {
  // Beğeni sayısını artır
  // API çağrısı yap
  console.log('Post liked:', postId);
};

<CardPost
  onLike={() => handleLike(postId)}
  likes={likeCount}
/>
```

### Yorum Fonksiyonu
```javascript
const handleComment = (postId) => {
  // Yorum sayfasına git
  navigation.navigate('Comments', { postId });
};

<CardPost
  onComment={() => handleComment(postId)}
  comments={commentCount}
/>
```

### Kaydet Fonksiyonu
```javascript
const handleSave = (postId) => {
  // Gönderiyi kaydedilenler listesine ekle
  savePost(postId);
};

<CardPost
  onSave={() => handleSave(postId)}
/>
```

```javascript
{posts.map(post => (

    key={post.id}
    name={post.name}
    time={post.time}
    text={post.text}
    type={post.type}
```


## SORUN GİDERME

### Yaygın Sorunlar

1. **Avatar görünmüyor**
   - Avatar path'ini kontrol edin
   - Varsayılan avatar otomatik yüklenir

2. **Callback fonksiyonları çalışmıyor**
   - Fonksiyonları doğru şekilde tanımladığınızdan emin olun
   - Undefined kontrolü yapın

3. **Karanlık tema çalışmıyor**
   - darkMode prop'unu true olarak ayarlayın
   - Colors constant'ının doğru import edildiğinden emin olun
