--------------------------------------------------------------------------------
1.2 PARAMETRELER (Props)
--------------------------------------------------------------------------------

┌─────────────────┬──────────────────────┬─────────────┬───────────────────┐
│ Parametre       │ Tip                  │ Varsayılan  │ Açıklama          │
├─────────────────┼──────────────────────┼─────────────┼───────────────────┤
│ text            │ string               │ -           │ Buton metni       │
│ type            │ 'primary'/'neutral'  │ 'primary'   │ Buton tipi        │
│ buttonStyle     │ 'default'/'outline'/ │ 'default'   │ Buton stili       │
│                 │ 'text'/'soft'        │             │                   │
│ size            │ 'default'/'sm'/'xs'  │ 'default'   │ Buton boyutu      │
│ mode            │ 'light'/'dark'       │ 'light'     │ Tema modu         │
│ disabled        │ boolean              │ false       │ Pasif durumu      │
│ onPress         │ function             │ -           │ Tıklama fonksiyonu│
│ hideArrow       │ boolean              │ false       │ Arrow'u gizle     │
└─────────────────┴──────────────────────┴─────────────┴───────────────────┘

--------------------------------------------------------------------------------
1.3 TİPLER (type)
--------------------------------------------------------------------------------

1. PRIMARY (Yeşil - Ana Butonlar)
   - Kullanım: Ana aksiyonlar için (Devam Et, Kaydet, Gönder)
   - Renk: #58B658 (Yeşil)
   
   Örnek:
   <Button 
     text="Devam Et"
     type="primary"
     onPress={handleNext}
   />

2. NEUTRAL (Gri - İkincil Butonlar)
   - Kullanım: İkincil aksiyonlar için (Geri, İptal)
   - Renk: #3C3E44 (Koyu Gri)
   
   Örnek:
   <Button 
     text="Geri"
     type="neutral"
     onPress={handleBack}
   />

--------------------------------------------------------------------------------
1.4 STİLLER (buttonStyle)
--------------------------------------------------------------------------------

1. DEFAULT (Dolu - Filled)
   - Arkaplan rengi dolu
   - Gölge efekti var
   - Ana butonlar için
   
   Örnek:
   <Button 
     text="Devam Et"
     buttonStyle="default"
     type="primary"
   />

2. OUTLINE (Çerçeveli)
   - Sadece kenarlık
   - Arkaplan şeffaf
   - İkincil aksiyonlar için
   
   Örnek:
   <Button 
     text="Geri"
     buttonStyle="outline"
     type="neutral"
   />

3. SOFT (Yumuşak)
   - Açık renkli arkaplan
   - Gölge yok
   - Hafif vurgular için
   
   Örnek:
   <Button 
     text="Daha Sonra"
     buttonStyle="soft"
     type="neutral"
   />

4. TEXT (Sadece Metin)
   - Arkaplan yok
   - Sadece metin
   - Linkler için
   
   Örnek:
   <Button 
     text="Giriş Yapın"
     buttonStyle="text"
     type="primary"
   />

--------------------------------------------------------------------------------
1.5 BOYUTLAR (size)
--------------------------------------------------------------------------------

1. DEFAULT (Varsayılan)
   - Yükseklik: 48px
   - Genişlik: 100% (Tam genişlik)
   - Font: 18px
   - Ana aksiyon butonları için
   
   Örnek:
   <Button 
     text="Devam Et"
     size="default"
   />

2. SM (Küçük)
   - Yükseklik: 40px
   - Genişlik: İçeriğe göre (Auto)
   - Font: 14px
   - İkincil butonlar için
   
   Örnek:
   <Button 
     text="Kaydet"
     size="sm"
   />

3. XS (Çok Küçük)
   - Yükseklik: 32px
   - Genişlik: İçeriğe göre (Auto)
   - Font: 14px
   - Küçük aksiyonlar için (Skip, Cancel, Link-gibi)
   
   Örnek:
   <Button 
     text="Skip"
     size="xs"
   />

3. RESPONSIVE TASARIM
================================================================================

Tüm componentler react-native-size-matters kütüphanesi ile responsive olarak
tasarlanmıştır:

- scale() → Genişlik için
- verticalScale() → Yükseklik için
- moderateScale() → Font boyutları için

Bu sayede farklı ekran boyutlarında (iPhone SE, iPhone 14, iPad, vs.) otomatik
olarak uyumlu görünür.


================================================================================
4. BEST PRACTICES (İyi Uygulamalar)
================================================================================

✅ YAPILMASI GEREKENLER:

1. Ana aksiyonlar için her zaman 'primary' type kullanın
2. İkincil aksiyonlar için 'neutral' type kullanın
3. Ekran genişliğindeki butonlar için 'default' size kullanın
4. Navigation butonları için 'xs' veya 'sm' size kullanın
5. disabled prop'u form validasyonu için kullanın
6. onPress fonksiyonunu her zaman tanımlayın

❌ YAPILMAMASI GEREKENLER:

1. Bir ekranda birden fazla 'primary' 'default' buton kullanmayın
2. Çok küçük (xs) butonları ana aksiyon için kullanmayın
3. 'text' style butonları kritik aksiyonlar için kullanmayın
4. Dark mode'u light ekranlarda karıştırmayın


================================================================================
5. ÖRNEK KULLANIM (Tam Ekran)
================================================================================

import React from 'react';
import { View, Text } from 'react-native';
import { StatusBar } from 'expo-status-bar';
import TopNavigation from '../components/TopNavigation';
import Button from '../components/Button';

const ExampleScreen = ({ navigation }) => {
  const handleBack = () => {
    navigation.goBack();
  };

  const handleSkip = () => {
    navigation.navigate('Home');
  };

  const handleNext = () => {
    navigation.navigate('NextScreen');
  };

  return (
    <View style={{ flex: 1, backgroundColor: '#FFFFFF' }}>
      <StatusBar barStyle="dark-content" backgroundColor="#fff" />
      
      {/* Top Navigation */}
      <TopNavigation
        showLeadingItem={true}
        showCenterItem={false}
        showTrailingItem={true}
        trailingType="button"
        trailingText="Skip"
        buttonType="neutral"
        buttonStyle="text"
        buttonSize="xs"
        onLeadingPress={handleBack}
        onTrailingPress={handleSkip}
      />
      
      {/* Content */}
      <View style={{ flex: 1, padding: 20 }}>
        <Text>İçerik buraya gelecek...</Text>
      </View>
      
      {/* Bottom Buttons */}
      <View style={{ padding: 20, gap: 16 }}>
        <Button 
          text="Devam Et"
          type="primary"
          buttonStyle="default"
          size="default"
          onPress={handleNext}
        />
        
        <Button 
          text="Geri"
          type="neutral"
          buttonStyle="outline"
          size="sm"
          onPress={handleBack}
        />
      </View>
    </View>
  );
};

export default ExampleScreen;
