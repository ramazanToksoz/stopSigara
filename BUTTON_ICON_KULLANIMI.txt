BUTTON LEFT ICON KULLANIMI
===========================

Button komponentine sol tarafta icon ekleme özelliği eklendi.

KULLANIM
--------

import Button from '../components/Button';

<Button
  text="Continue with Google"
  type="neutral"
  buttonStyle="soft"
  size="default"
  leftIcon={require('../assets/images/icons/Google_icon.png')}
  hideArrow={true}
  onPress={handlePress}
/>


YENİ ÖZELLIK
------------

leftIcon (image source)
  - Sol tarafta gösterilecek icon
  - require() ile image import edilmeli
  - Örnek: leftIcon={require('../assets/images/icons/icon.png')}
  - İkon boyutu otomatik ayarlanır:
    * default size: 20x20
    * sm/xs size: 18x18

NOTLAR
------
- leftIcon ve arrow (sağ ok) birlikte kullanılabilir
- hideArrow={true} ile sağ ok gizlenebilir
- Icon rengi otomatik tintColor almaz, orijinal renkli gösterilir
- SSO (Social Sign-On) butonları için idealdir
